Teoria bilingue

Conicas principales

Circunferencia

Todos los puntos a la misma distancia de un centro

La circunferencia es el conjunto de puntos del plano que estan a una distancia fija, llamada radio, de un punto fijo llamado centro. Su ecuacion canonica es (x - h)^2 + (y - k)^2 = r^2.

A circle circumference is the set of all points in the plane at a fixed distance, the radius, from a fixed point called the center. Its standard equation is (x - h)^2 + (y - k)^2 = r^2.

Circulo

La region interior limitada por una circunferencia

El circulo incluye todos los puntos interiores y la frontera de una circunferencia. Su area es A = pi r^2 y la longitud de su frontera es L = 2 pi r.

A disk is the region made of all interior points and the boundary circle. Its area is A = pi r^2, and the boundary length is L = 2 pi r.

Parabola

Puntos equidistantes de un foco y una directriz

La parabola es el conjunto de puntos que estan a la misma distancia de un punto fijo, el foco, y de una recta fija, la directriz. Si el vertice es (h,k), una forma comun es (x - h)^2 = 4p(y - k).

A parabola is the set of points equidistant from a fixed point, the focus, and a fixed line, the directrix. With vertex (h,k), a common form is (x - h)^2 = 4p(y - k).

Elipse

La suma de distancias a dos focos permanece constante

La elipse es el conjunto de puntos cuya suma de distancias a dos focos es constante. Para centro (h,k) y eje mayor horizontal: ((x - h)^2 / a^2) + ((y - k)^2 / b^2) = 1.

An ellipse is the set of points whose sum of distances to two foci is constant. For center (h,k) and a horizontal major axis: ((x - h)^2 / a^2) + ((y - k)^2 / b^2) = 1.

Hiperbola

La diferencia de distancias a dos focos permanece constante

La hiperbola es el conjunto de puntos cuya diferencia absoluta de distancias a dos focos es constante. Una forma horizontal es ((x - h)^2 / a^2) - ((y - k)^2 / b^2) = 1, con asintotas que orientan sus ramas.

A hyperbola is the set of points whose absolute difference of distances to two foci is constant. A horizontal form is ((x - h)^2 / a^2) - ((y - k)^2 / b^2) = 1, with asymptotes guiding the branches.

Herramientas TIC por integrante

Modelar y graficar conicas

Integrante 1: GeoGebra

Permite escribir ecuaciones directamente y manipular parametros con deslizadores.

  1. Abrir la calculadora grafica.
  2. Escribir una ecuacion como x^2 + y^2 = 25.
  3. Crear deslizadores para h, k y r.
  4. Observar centro, radio, focos o asintotas segun la conica.
Abrir GeoGebra

Integrante 2: Desmos

Es util para graficar rapidamente ecuaciones y comparar varias curvas en el mismo plano.

  1. Abrir la calculadora grafica.
  2. Ingresar ((x-1)^2/9)+((y+2)^2/4)=1.
  3. Agregar restricciones o colores para diferenciar conicas.
  4. Usar la captura como evidencia visual en la wiki.
Abrir Desmos

Integrante 3: WolframAlpha

Identifica conicas, calcula propiedades y muestra pasos o informacion complementaria.

  1. Abrir WolframAlpha.
  2. Escribir una consulta como hyperbola x^2/9 - y^2/4 = 1.
  3. Revisar grafica, vertices, focos y asintotas.
  4. Comparar el resultado con el desarrollo manual.
Abrir WolframAlpha

Ejercicios resueltos por conica

Conica Ejercicio Solucion breve
Circunferencia Hallar centro y radio de (x - 2)^2 + (y + 1)^2 = 16. Centro (2,-1) y radio 4, porque r^2 = 16.
Circulo Calcular el area de un circulo de radio 5. A = pi r^2 = 25pi unidades cuadradas.
Parabola Identificar vertice y foco de (x - 1)^2 = 8(y + 2). 4p = 8, entonces p = 2. Vertice (1,-2) y foco (1,0).
Elipse Analizar x^2/25 + y^2/9 = 1. Centro (0,0), a = 5, b = 3, c = 4; focos (-4,0) y (4,0).
Hiperbola Analizar x^2/9 - y^2/4 = 1. Centro (0,0), a = 3, b = 2, c = sqrt(13); asintotas y = +/- 2x/3.
