Cónicas principales

Circunferencia

Todos los puntos a la misma distancia de un centro

La circunferencia es el conjunto de puntos del plano que están a una distancia fija, llamada radio, de un punto fijo llamado centro. Su ecuación canónica es (x - h)^2 + (y - k)^2 = r^2.

A circle circumference is the set of all points in the plane at a fixed distance, the radius, from a fixed point called the center. Its standard equation is (x - h)^2 + (y - k)^2 = r^2.

Círculo

La región interior limitada por una circunferencia

El círculo incluye todos los puntos interiores y la frontera de una circunferencia. Su área es A = π r² y la longitud de su frontera es L = 2π r.

A disk is the region made of all interior points and the boundary circle. Its area is A = π r², and the boundary length is L = 2π r.

Parábola

Puntos equidistantes de un foco y una directriz

La parábola es el conjunto de puntos que están a la misma distancia de un punto fijo, el foco, y de una recta fija, la directriz. Si el vértice es (h,k), una forma común es (x - h)² = 4p(y - k).

A parabola is the set of points equidistant from a fixed point, the focus, and a fixed line, the directrix. With vertex (h,k), a common form is (x - h)^2 = 4p(y - k).

Elipse

La suma de distancias a dos focos permanece constante

La elipse es el conjunto de puntos cuya suma de distancias a dos focos es constante. Para centro (h,k) y eje mayor horizontal: ((x - h)^2 / a^2) + ((y - k)^2 / b^2) = 1.

An ellipse is the set of points whose sum of distances to two foci is constant. For center (h,k) and a horizontal major axis: ((x - h)^2 / a^2) + ((y - k)^2 / b^2) = 1.

Hipérbola

La diferencia de distancias a dos focos permanece constante

La hipérbola es el conjunto de puntos cuya diferencia absoluta de distancias a dos focos es constante. Una forma horizontal es ((x - h)² / a²) - ((y - k)² / b²) = 1, con asíntotas que orientan sus ramas.

A hyperbola is the set of points whose absolute difference of distances to two foci is constant. A horizontal form is ((x - h)^2 / a^2) - ((y - k)^2 / b^2) = 1, with asymptotes guiding the branches.

Herramientas TIC por integrante

Modelar y graficar cónicas

Integrante 1: GeoGebra

Permite escribir ecuaciones directamente y manipular parámetros con deslizadores.

  1. Abrir la calculadora gráfica.
  2. Escribir una ecuación como x² + y² = 25.
  3. Crear deslizadores para h, k y r.
  4. Observar centro, radio, focos o asíntotas según la cónica.
Abrir GeoGebra

Integrante 2: Desmos

Es útil para graficar rápidamente ecuaciones y comparar varias curvas en el mismo plano.

  1. Abrir la calculadora gráfica.
  2. Ingresar ((x-1)²/9)+((y+2)²/4)=1.
  3. Agregar restricciones o colores para diferenciar cónicas.
  4. Usar la captura como evidencia visual en la wiki.
Abrir Desmos

Integrante 3: WolframAlpha

Identifica cónicas, calcula propiedades y muestra pasos e información complementaria.

  1. Abrir WolframAlpha.
  2. Escribir una consulta como hyperbola x²/9 - y²/4 = 1.
  3. Revisar gráfica, vértices, focos y asíntotas.
  4. Comparar el resultado con el desarrollo manual.
Abrir WolframAlpha

Ejercicios resueltos por cónica

Cónica Ejercicio Solución breve
Circunferencia Hallar centro y radio de (x - 2)² + (y + 1)² = 16. Centro (2,-1) y radio 4, porque r² = 16.
Círculo Calcular el área de un círculo de radio 5. A = πr² = 25π unidades cuadradas.
Parábola Identificar vértice y foco de (x - 1)² = 8(y + 2). 4p = 8, entonces p = 2. Vértice (1,-2) y foco (1,0).
Elipse Analizar x²/25 + y²/9 = 1. Centro (0,0), a = 5, b = 3, c = 4; focos (-4,0) y (4,0).
Hipérbola Analizar x²/9 - y²/4 = 1. Centro (0,0), a = 3, b = 2, c = √13; asíntotas y = ±2x/3.
